Scientific Method

The scientific method is a process used to find the answers to scientific problems. The process is as follows: Problem, Investigation, Hypothesis, Experiment, Results, and Conclusion. These steps are best remembered through the acronym PIHERC, which can also stand for "Pigs In Hawaii Eat Raw Corn." When following the Scientific Method, do not forget to go through every step thouroughly.

Problem

A scientific problem is always in the form of a question, and must be testable, as in it can be answered by conducting an experiment. For example, "Which types of Scotch brand tape stick best?" A problems statement will contain the independent variable, the value you change, and the dependent variable A problem statement also has to be specific, so as you'll see in the example, I explicitly stated the tape was from Scotch. You will always find the independent variable and dependent variable in a problem statement in that order. In this example, the indenependent and dependent variables are bolded: "Which types of Scotch tape stick best?"
